                                                                    Palash Nandi a year ago

                                                                    Hi Guys,
                                                                    If you are getting a blank screen make sure

                                                                    1. The script include statements are placed at the end of the <body> tag after the templates definitions and also
                                                                    2. The handlebar.js should be before the app.js.

                                                                    Point 1 is critical and easy to overlook as app.js needs the body elements and they need to be defined before the scripts are run. Hence because of the same reason we cannot move the script includes to the <head> section as is generally the norm. Hope this solves the blank screen issue :)
